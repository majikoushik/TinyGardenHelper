# Roadmap

## MVP Implementation Roadmap

### Milestone 1: Unity Project Foundation
- Create folder structure and essential documentation.
- Establish architecture guidelines and safety checklists.

### Milestone 2: Core Systems
- Implement `GameManager`, `SceneLoader`, `SaveSystem`, `AudioManager`, and `UIManager`.
- Add local save/load logic and audio toggles.

### Milestone 3: Boot, Main Menu, and Garden Hub
- Create Boot, Splash, Main Menu, Garden, and Settings scenes.
- Add safe area handling and basic transitions.
- Build garden background with three tappable plant prefabs.

### Milestone 4: Color Matching Mini-Game
- Implement large answer buttons and color matching logic.
- Add gentle try-again feedback.
- Wire completion back to garden plant growth.

### Milestone 5: Counting Fruits Mini-Game
- Show 1-5 fruit sprites and number choices.
- Implement counting logic and completion feedback.

### Milestone 6: Shape Sorting Mini-Game
- Add drag/drop or tap-to-place shapes (circle, square, triangle).
- Snap correct shapes into matching slots.

### Milestone 7: Rewards And Polish
- Add celebration views and sparkles.
- Add animal friend unlock sequence.
- Integrate placeholder voice and SFX hooks.

### Milestone 8: Mobile Hardening
- Test on various screen sizes and safe areas.
- Profile memory and ensure frame pacing.
- Build Android (APK/AAB) and iOS (Xcode) development builds.

---

## Post-MVP Ideas (Future Scope)
- **More Mini-Games**: Tracing letters/numbers, simple jigsaw puzzles.
- **More Gardens**: Different themes (e.g., Winter Garden, Desert Garden).
- **More Animals**: Unlock a whole collection of animal friends.
- **Sensory-Safe Mode**: Dedicated settings for reduced animation and softer sounds.
